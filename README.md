# Snake — простая онлайн-игра

Минимальная реализация игры «Snake» (HTML/CSS/JS). Можно разместить на GitHub и запустить через Render как Static Site.

## Как запустить
1. Создай репозиторий на GitHub и загрузи папку snake-game.
2. На Render создай новый проект (Static Site) и подключи репозиторий.
3. Build Command — ничего, Publish Directory — /.
4. После деплоя сайт будет доступен по URL Render.
